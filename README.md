# AN END-TO-END STUDY ON PROMPT ENGINEERING TECHNIQUES IN GENERATIVE AI FOR AUTOMATED COURSE UNDERSTANDING, SKILL EXTRACTION AND PERSONALIZED LEARNING INSIGHTS USING LARGE-SCALE ONLINE EDUCATION DATA

This repository has been created for the Master’s thesis work completed at Liverpool John Moores University (LJMU).  
The code has been written in colab Notebook form, and the work has been grouped into separate experiment folders for easy reading and testing.

## Project Overview

This project has been used to study how large language models can be used for automated course understanding.  
Different prompt styles and model settings have been tested across several tasks.

The main aim of the project has been to check how well language models can:

- summarise course content
- extract skills from course data
- predict course difficulty
- generate learning insights
- compare results across models and prompts

All experiments have been arranged so the workflow can be followed step by step.

## What Is Included

The repository has been prepared for thesis experimentation and result analysis.

## How to Run the Project

The notebooks have been designed to be run in order.

1. The repository should be cloned or downloaded.
2. Jupyter Notebook or JupyterLab should be opened.
3. The preprocessing notebook should be run first.
4. The task notebooks should then be run one by one.
5. The results notebook should be used at the end for comparison and analysis.
